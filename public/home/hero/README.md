# Hero Slider Images

Place hero slider images in this directory:

## Required Files

- `slide-1.jpg` - Main hero image (1920x1080px, premium bag photo, dark/moody lighting)
- `slide-2.jpg` - Collection showcase (1920x1080px, multiple bags, lifestyle setting)
- `slide-3.jpg` - Detailed product shot (1920x1080px, close-up of premium bag detail)

## Image Specifications

- **Format**: JPG (optimized for web)
- **Dimensions**: 1920x1080px (16:9 aspect ratio)
- **File size**: Max 300KB per image (use compression)
- **Style**: Dark, moody, premium aesthetic with black/gray tones
- **Overlay**: Images should work well with dark gradient overlay
